# البورتفوليو المهني — نسخة محدّثة وفق الفهرس الأصلي

موقع ويب يعرض كامل محتويات البورتفوليو المهني للسيدة نجلاء الساحلي،
بمحتوى منقول حرفيًّا من الوثيقة الأصلية، ومنظّم وفق الفهرس الذي اعتمدته المؤلِّفة.

## 🆕 التحديثات في هذه النسخة

- **الخط الأسفل** في رأس الصفحات الفرعيّة أصبح مستقيمًا (border-bottom وردي) ولم يعد يحجب الكتابة.
- **محتوى الندوات والوحدات** منقول من الوثيقة الأصليّة دون تغيير، للأمانة العلميّة.
- **التكوين النظري** يحتوي الآن 6 أبعاد وفق الفهرس:
  1. البعد الأخلاقي القيمي (3 ندوات)
  2. البعد المؤسّسي التنظيمي (3 ندوات + محاضرتان)
  3. البعد المعرفي (6 وحدات)
  4. **البعد الذاتي النفسي** (جديد - ندوتان: البورتفوليو + التحليل التأمّلي)
  5. البعد الاجتماعي التواصلي (ندوتان)
  6. البعد الرقمي (3 وحدات)
- **التكوين الميداني** منظَّم وفق الفهرس إلى **4 مجالات**:
  1. **مجال التدقيق المدرسي** (تقييم الفاعلين + تقييم المؤسسة + اللجان)
  2. **القيادة والتواصل: مشروع الدائرة** (الذكاء الاصطناعي + تثمين المبادرات)
  3. **مجال البحث والتجديد** (البحث التربوي + مبادرة التجديد)
  4. **مجال هندسة التكوين** (التكوين من الملاحظة إلى الاستقلاليّة)

## 📂 الصفحات الرئيسيّة

| الصفحة | الدور |
|------|-------|
| `index.html` | **الاستقبال** — Hero + إحصائيّات + الأبعاد + المجالات + معرض الصور |
| `theorique.html` | **التكوين النظري** — قائمة الأبعاد الستّة |
| `terrain.html` | **التكوين الميداني** — قائمة المجالات الأربعة |
| `recherches.html` | **الأبحاث التربويّة** — بطاقات بحوث تفصيليّة |
| `projets.html` | **المشاريع البيداغوجيّة** — بطاقات مشاريع |
| `cv.html` | **السيرة الذاتيّة** (قابلة للتعديل) |

## 📂 صفحات فرعيّة

| الصفحة | الدور |
|------|-------|
| `dimension.html?id=...` | بُعد محدّد ووحداته (6 أبعاد) |
| `unit.html?key=...` | محتوى وحدة كامل (يعمل للأبعاد والمجالات) |
| `mjal.html?id=...` | مجال ميداني محدّد وأقسامه |

## 📊 المحتوى

- **17 وحدة/ندوة** في التكوين النظري — كل المحتوى منقول من الوثيقة الأصلية
- **8 أقسام** في التكوين الميداني موزّعة على 4 مجالات
- **~835 KB** من المحتوى المنظّم في HTML

## 🌐 طرق التشغيل

### 1) محلّيًا
* افتح `index.html` بنقرة مزدوجة.

### 2) Netlify (الأنسب لـ Google Sites)
1. اذهب إلى [netlify.com/drop](https://app.netlify.com/drop).
2. اسحب مجلّد `site/` كاملًا.
3. ستحصلين على رابط مباشر فورًا.
4. للإدماج في Google Sites: **Insert → Embed → By URL**.

## ✏️ التعديل والإضافة

كلّ المحتوى مركّز في ملفّين:
- `data.js` — بنية الموقع (الأبعاد، الوحدات، السيرة الذاتيّة، الأبحاث، المشاريع، معرض الصور)
- `content_data.js` — المحتوى التفصيلي (HTML لكل وحدة)
